# Analisis de Texto

Hola Bienvenido a este repositorio, en esta ocasion vamos a realizar un pequeño ejercicio de analisis de sentimientos tomando como base un dataframe de comentarios en videos de Youtube

Este es un proyecto de Análisis de Datos por lo cual no vamos a implementar modelos de ML

Vamos a aprender a usar algunas librerias interesantes como TextBlob, WordCloud y STOPWORDS para manipular nuestro dataset y tener una apreciacion mas visual de nuestros datos
